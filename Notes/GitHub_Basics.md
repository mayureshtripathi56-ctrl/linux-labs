# GitHub Basics
# GitHub Fundamentals

## What is GitHub?

GitHub is a cloud platform that hosts Git repositories and enables developers to collaborate, review code, manage projects, and maintain secure backups.

---

## Why Do Companies Use GitHub?

Companies use GitHub because it allows multiple developers to:

- Collaborate on the same project
- Track every change
- Review code before merging
- Restore previous versions
- Maintain project history
- Manage releases

---

## What is a Repository?

A Repository (Repo) is a project folder that is tracked by Git.

Example:

linux-labs/

README.md

Notes/

Projects/

Screenshots/

---

## Git vs GitHub

Git

- Software
- Installed locally
- Tracks project history
- Works offline

GitHub

- Cloud platform
- Stores Git repositories
- Enables collaboration
- Requires internet
