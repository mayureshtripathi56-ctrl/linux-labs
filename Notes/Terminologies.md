# Important Terminologies

## Git

Distributed Version Control System

---

## GitHub

Cloud platform for Git repositories

---

## Repository

Project folder tracked by Git

---

## Commit

Saved snapshot of a project

---

## Push

Upload local commits to GitHub

---

## Pull

Download the latest changes from GitHub

---

## Clone

Download an existing GitHub repository to your local computer

---

## Branch

An independent line of development within a project

---

## Merge

Combining changes from one branch into another
